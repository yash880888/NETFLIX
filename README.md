# 🎬 Netflix Clone

A Netflix Landing Page Clone built using **HTML and CSS**.  
This project recreates the Netflix homepage design and focuses on layout styling and responsive web design concepts.

🔗 Live Demo:  
https://yash880888.github.io/NETFLIX/

---

## 📌 Project Overview

This project is a clone of the Netflix homepage created for learning and practicing frontend development skills.

It includes:
- Netflix-style homepage layout
- Hero section
- Movie and TV show sections
- FAQ section
- Footer design

The website is **responsive**, but **not fully responsive** for all screen sizes and devices.

---

## 🚀 Technologies Used

- HTML5

- Flexbox
- Media Queries
